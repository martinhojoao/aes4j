## AES for Java

Este repositório contém um algoritmo (`Aes4J.java`) que implementa o Advanced Encryption Standard em Java sem o auxílio de bibliotecas externas. 
Foi a minha solução, em parceria com o [@FelipeBona](https://github.com/FelipeBona), para um trabalho da faculdade e recebeu a nota máxima. 
Eu fiquei tão satisfeito com o resultado que decidi guardá-lo aqui no GitHub.

Para referência, o arquivo `AesCipher.java` contém a solução equivalente usando a biblioteca `javax.crypto.Cipher`.
